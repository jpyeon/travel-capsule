# Packing Assistant

A web app for planning what to pack. You add your clothes to a digital closet, create a trip with dates and activities, and the app figures out a reasonable capsule wardrobe, suggests daily outfits, and spits out a packing checklist. Eventually there'll be AI-generated outfit previews too.

## How it'll work

- Add clothes to your closet
- Create a trip (destination, dates, activities)
- Get a suggested capsule wardrobe for the trip
- See daily outfit suggestions
- Get a packing checklist
- (Planned) AI-generated outfit previews via Gemini / Nano Banana

## Status

Early development. Currently building out the backend architecture and core algorithms. No UI yet.

## Tech

- React + Vite
- TypeScript
- Supabase
- Feature-based architecture
